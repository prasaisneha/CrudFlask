Flask is a web application framework also referred to as a microframework of python. It is designed to keep the core of the application simple and scalable.

Flask is based on the Werkzeg WSGI toolkit and the Jinja2 template engine.  It is lightweight and its modular design makes it easier to adapt. The main feature of the flask is a built-in development server, fast debugger, integrated support for unit testing, secure cookie, etc.
